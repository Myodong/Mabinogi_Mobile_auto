- **원인/배경**: 어비스 커스텀 리스트는 항목마다 방식(혼자하기/함께하기)과 매칭(우연한 만남/

- **수정 (`mabinogi_gui.ps1`)**

  - `Get-AbyssMatchingIssues` 신설: 첫 항목과 방식/매칭이 다른 항목 목록 `[{Index; Mode; Matching;
    Reason}]` 반환 (`return $issues` + 호출부 `@()` 규약).
  - `Update-AbyssInputLock` 신설: 잠금 시 라디오 5종(`$rbAcrSolo/$rbAcrParty/$rbAcrChance/
    $rbAcrFindParty/$rbAcrPartyLead`) `Enabled=$false` + 리스트 값으로 `Checked` 보정
    (`$script:crLoading` 가드로 저장 이벤트 억제, `$script:acrLockUpdating` 로 재진입 차단).
  - **잠긴 이유 안내(툴팁) — 실기 2회 조정**: WinForms 는 비활성 컨트롤에 마우스 이벤트를 주지

    오갈 때 **깜박였고**, ②패널에만 걸었더니 잠긴 라디오 위에서는 **아예 안 떴다**.
    → 최종: 패널 `MouseMove` 에서 커서가 어느 비활성 라디오 `Bounds` 안인지 직접 판정해
    `$toolTip.Show(...)` 로 띄우고, **같은 컨트롤에서는 재호출하지 않아** 깜박임이 없다
    (`$script:acrTipShownFor` 로 추적, `MouseLeave`·잠금 해제 시 `Hide`). 자동부활 체크박스와
    같은 사용감 확보. 항상 보이는 안내 라벨도 시도했으나 툴팁으로 충분해 사용자 요청으로 제거.
